
# 需求

基于命令行的记账软件

# 目标
1. 运用 C++ 知识
2. 软件设计

# 整体描述

1. 功能
- 记账
    + 记录每笔收入和支出
- 查询
    + 查询当前所有账目, 统计总收支;
    + 查询全部账目、查询收入、查询支出
- 数据需要做持久化保存（可以保存在文本文件中）
- 软件采取分级菜单形式
    + 每一级菜单有 "返回主菜单" 功能
    
2. 主菜单界面 3 个选项
- 1. 记账
    + 收入
    + 支出
    + 返回主菜单
- 2. 查询
    + 统计所有账目
    + 统计收入
    + 统计支出
    + 返回主菜单
- 3. 退出
    + 确认退出? Y: 退出, N: 不退出
    
    
3. 程序流程
> 1. 加载数据
> 2. 整体是一个循环, 在循环内部显示主菜单, 读取键盘输入确定用户的操作
